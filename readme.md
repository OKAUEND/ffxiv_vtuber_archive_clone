# FFXIV Vtuber Archives

Youtube で主に FFXIV のプレイ及び配信で活動中の Vtuber の一覧及び過去の配信一覧

## Description

Youtube で主に FFXIV のプレイ及び配信で活動中の Vtuber の一覧及び過去の配信一覧を、
Youtube Date API を用いて取得し、表示する。

## Install

### Step.1

```
yarn install
```

## Local Preview

```
yarn serve
```

## Project Build

```
yarn Build
```
